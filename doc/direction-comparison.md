# 《午夜验客》视觉方向 A / B / C 比较

## 评审方法

- 三个方向使用同一位顾客、同一对白、同一线索位置、同一判断按钮与同一 390×844 视口。
- 每个方向都检查营业默认态与夜客揭晓态。
- A 为当前通过垂直切片 QA 的基线；B/C 是通过 `?style=b`、`?style=c` 启用的方向原型。
- 截图位于 `_qa/directions/`，生成素材来源记录位于 `doc/direction-variant-generation.json`。

## 同状态评分

| 维度 | A 荧光小票黑色电影 | B 失真监控录像 | C 午夜怪谈漫画 |
|---|---:|---:|---:|

## A · 荧光小票黑色电影

- 优点：信息最清楚，顾客表情可信，小票与收银设备 UI 融合成熟，制作风险最低。
- 缺点：真人便利店怪谈较接近影视常见语言，游戏自身的视觉符号不如 B/C 强。
- 最适合：希望推理清晰、情绪克制、角色表演自然的版本。
- 证据：`_qa/directions/a-game-390x844.png`、`a-reveal-390x844.png`。

## B · 失真监控录像

- 优点：鱼眼边缘、扫描线、时间码和延迟残影让“玩家正在监视异常”这一身份立即成立；揭晓最令人不安。
- 缺点：高机位使顾客与商品变小，扫描线降低文字与细节对比；连续 8 位顾客都保持故障画面会产生视觉疲劳。
- 扩展风险：每位顾客的关键线索必须比 A/C 更大、更简单；生成素材需要持续锁定同一监控镜位，否则墙面和柜台会跳动。
- 最适合：更偏严肃惊悚、观察压力和监控叙事的版本。
- 证据：`_qa/directions/b-game-390x844.png`、`b-reveal-390x844.png`。

## C · 午夜怪谈漫画

- 优点：角色、热点、对白框和按钮都能使用同一粗墨线与网点语言；酸黄色线索和红色判断键在手机上非常直接；最能承载怪客的荒诞反应。
- 缺点：现实恐惧感弱于 B，若每次揭晓都使用巨大爆炸线会变得吵闹，需要把破格和冲击线保留给关键时刻。
- 扩展风险：12 位人物需要统一线宽、网点密度和有限色盘；比监控镜位更容易通过固定 reference 保持一致。
- 最适合：最初承诺的“诡异但好笑”，并且更像一款有独立品牌的游戏。
- 证据：`_qa/directions/c-game-390x844.png`、`c-reveal-390x844.png`。

## 建议

推荐选择 C 作为完整游戏的基础方向，并只借用 B 的“一帧横向信号撕裂”作为夜客揭晓动效。这样不把常态画面做成低可读性的监控滤镜，同时保留一次令人不安的异常信号。

如果目标从“诡异但好笑”改成“持续压迫、偏严肃恐怖”，则选择 B，并重新放大线索热点与顾客主体。
